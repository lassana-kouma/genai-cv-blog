Computer Vision & Generative AI

Rapport académique — Analyse des modèles de perception et génération visuelle

1. Introduction

La Computer Vision vise à transformer une image (matrice de pixels) en information sémantique exploitable. Les approches modernes reposent sur trois étapes : extraction de caractéristiques, représentation, et prédiction.

Image ∈ ℝ^(H × W × C)
📌 Schéma à insérer : Pipeline global Vision (Image → Features → Model → Output)

2. Fondations de la vision artificielle

Problème central

Une image brute contient uniquement des valeurs numériques. Le défi est de construire une représentation robuste invariant aux transformations (rotation, échelle, illumination).

Analogie : une image est un puzzle de pixels. Le modèle doit comprendre l’image complète à partir des pièces.

3. Feature Engineering (pré-deep learning)

SIFT

Détection de points clés invariants aux transformations.

Étapes :

  • Détection de keypoints
  • Description locale
  • Vecteur de signature
📌 Schéma SIFT : Keypoints → Patch → Descripteur

HOG

Représentation basée sur les gradients d’intensité.

Principe : distribution des orientations des bords.

📌 Schéma HOG : Image → Gradients → Histogrammes

4. CNN (Convolutional Neural Networks)

Principe

Les CNN apprennent automatiquement des représentations hiérarchiques : bords → textures → objets.

Feature map = Image * Kernel

Composants

  • Convolution
  • ReLU
  • Pooling
  • Fully Connected
Analogie : un CNN est un système de filtres successifs qui simplifient progressivement une image complexe.
📌 Schéma CNN : Input → Conv → ReLU → Pool → Dense

5. Generative AI en Vision

GAN

Modèle basé sur une compétition entre générateur et discriminateur.

min_G max_D V(D,G)
📌 Schéma GAN

VAE

Modèle probabiliste basé sur un espace latent régularisé.

Loss = reconstruction + KL divergence
📌 Schéma VAE

Diffusion Models

Processus de débruitage progressif.

📌 Schéma Diffusion (forward / reverse)

6. Comparaison des modèles

Modèle Principe Forces Limites
GAN Adversarial learning Images réalistes Instabilité
VAE Probabiliste Espace structuré Flou
Diffusion Denoising SOTA qualité Lent

7. Conclusion

La Computer Vision moderne repose sur l’évolution des représentations : de l’ingénierie manuelle (SIFT/HOG) vers l’apprentissage profond (CNN), et aujourd’hui vers les modèles génératifs (GAN, VAE, Diffusion).

Les modèles génératifs permettent non seulement de comprendre les données, mais aussi de modéliser leur distribution complète.
