01. Architecture Adversariale (GAN)

L'architecture GAN repose sur un système de feedback constant. Le Générateur ne voit jamais les vraies images ; il n'apprend que par les "reproches" du Discriminateur via la rétropropagation du gradient.

// Objective Function
L = E[log D(x)] + E[log(1 - D(G(z)))]

Figure 1 : Flux de compétition entre G(z) et D(x).

02. Inférence Variationnelle (VAE)

Figure 2 : Le goulot d'étranglement (Bottleneck) probabiliste.

Dans un VAE, l'image est projetée dans un espace latent régularisé. Le schéma met en évidence que nous ne stockons pas une valeur fixe, mais des paramètres statistiques (μ, σ) permettant d'échantillonner de nouvelles données.

// Reparameterization Trick
z = μ + σ ⊙ ε where ε ~ N(0,1)

03. Processus de Diffusion Itératif

Le modèle de diffusion se distingue par sa temporalité. Il ne s'agit pas d'une passe unique, mais d'une succession de micro-étapes de débruitage.

Figure 3 : Chaîne de Markov du processus de diffusion (Ho et al., 2020).

Chaque étape $t$ du processus inverse tente de prédire le bruit ajouté à l'étape $t-1$ pour restaurer progressivement la structure de l'image.
