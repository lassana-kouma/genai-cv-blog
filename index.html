Generative AI en Computer Vision

Une explication simple (ELI5) mais rigoureuse des modèles GAN, VAE et Diffusion

1. C’est quoi le Generative AI en Computer Vision ?

Le Generative AI en vision par ordinateur est une branche de l’intelligence artificielle qui vise à modéliser la distribution des données visuelles afin de générer de nouvelles images réalistes.

Contrairement aux modèles discriminatifs (classification, détection), les modèles génératifs apprennent la structure interne des données : formes, textures, relations spatiales, etc.

Apprentissage : Images réelles → Modèle → Distribution P(x) Génération : bruit z → Modèle → Image synthétique

Mathématiquement, le but est d’approximer une distribution inconnue P(x) (images réelles) par une distribution générée Pθ(x).

Imagine que tu regardes des milliers de chats 🐱. Au bout d’un moment, ton cerveau comprend “ce qu’est un chat” et peut en dessiner un nouveau sans copier.

Exemple concret

  • Créer un visage humain qui n’existe pas
  • Transformer un croquis en image réaliste
  • Compléter une image partiellement détruite

2. Pourquoi utiliser le Generative AI ?

1. Data Augmentation

Générer de nouvelles données pour entraîner des modèles lorsque les données sont rares ou coûteuses.

Comme inventer des exercices supplémentaires pour mieux réviser.

2. Simulation (Médecine, autonome)

Simuler des cas rares (tumeurs, accidents) pour entraîner des systèmes robustes.

3. Création artistique

Utilisé dans le design, le cinéma et les jeux vidéo.

Une IA peut devenir un artiste 🎨

4. Image-to-Image

Transformer une image en une autre (jour → nuit, sketch → photo).

Pourquoi c’est révolutionnaire ?

  • Permet de créer sans données réelles
  • Réduit le coût de collecte de données
  • Ouvre la voie à des systèmes créatifs

3. Architectures principales

GAN (Generative Adversarial Networks)

Introduits par Goodfellow (2014), les GAN reposent sur un jeu à somme nulle entre deux réseaux :

  • Générateur (G) : produit des images
  • Discriminateur (D) : distingue vrai/faux
z (bruit) → G → image fake → D → probabilité réel/fake
Un faussaire fabrique des faux billets 💸 Un policier essaye de les détecter → Les deux deviennent meilleurs

Explication plus académique

Le modèle optimise une fonction min-max :

min_G max_D E[log D(x)] + E[log(1 - D(G(z)))]

Avantages

  • Images très réalistes
  • Rapide à générer

Limites

  • Instabilité d’entraînement
  • Mode collapse (manque de diversité)

VAE (Variational AutoEncoder)

Les VAE apprennent une représentation probabiliste appelée espace latent.

Image → Encodeur → (μ, σ) → z → Décodeur → Image
Tu compresses une photo → puis tu la reconstruis Comme zip → unzip

Explication académique

Le VAE maximise une borne ELBO :

ELBO = Reconstruction Loss + KL Divergence

Avantages

  • Stable
  • Espace latent interprétable

Limites

  • Images souvent floues

Diffusion Models

Les modèles de diffusion apprennent à inverser un processus de bruit.

Image → bruit progressif → bruit total → modèle → reconstruction
Tu prends une photo → tu la salis progressivement Puis tu apprends à la nettoyer étape par étape

Explication académique

Processus en deux phases :

  • Forward : ajouter du bruit
  • Reverse : enlever le bruit

Avantages

  • Qualité SOTA
  • Très stable

Limites

  • Lent (plusieurs étapes)
  • Coût computationnel élevé

4. Comparaison approfondie

Modèle Idée Analogie Avantages Limites
GAN Jeu adversarial Faussaire vs policier Images réalistes, rapide Instable, manque diversité
VAE Compression probabiliste Résumé puis reconstruction Stable, interprétable Flou
Diffusion Bruit → nettoyage Nettoyer une image sale Qualité SOTA Lent, coûteux

Analyse critique

Chaque modèle correspond à une philosophie différente :

  • GAN : compétition
  • VAE : probabilité
  • Diffusion : processus physique (bruit)
C’est comme 3 façons d’apprendre à dessiner :
  • GAN → compétition
  • VAE → comprendre et reconstruire
  • Diffusion → corriger progressivement

Conclusion

Le Generative AI représente aujourd’hui une des avancées majeures en intelligence artificielle.

Les modèles de diffusion dominent actuellement, mais les GAN restent utiles pour leur rapidité, et les VAE pour leur structure mathématique.

Il n’y a pas un modèle parfait. Chaque outil est adapté à un problème différent.
